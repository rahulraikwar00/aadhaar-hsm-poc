Key Features

🔑

Tokenization

Sensitive Aadhaar data replaced with secure tokens (T-UUID format). Raw data never stored.

🔐

HSM Encryption

Hardware Security Module integration for cryptographic operations and key management.

👁️

Field-level Masking

Automatic data masking for Aadhaar numbers, emails, and phone numbers.

📋

Audit Logging

Complete audit trail of all vault operations with PostgreSQL storage.

✓

Input Validation

Aadhaar format validation, email RFC 5322 compliance, phone number checks.

📊

Prometheus Metrics

Built-in metrics for store, retrieve, delete operations with Grafana support.

Quick Start

Run the vault with Docker Compose in seconds…

Clone & Run
# Clone and start all services
git clone https://github.com/rahulraikwar00/aadhaar-hsm-poc.git
cd aadhaar-hsm-poc
docker compose up -d
Store Aadhaar Data
curl -X POST http://localhost:8000/vault/store \
  -H "Content-Type: application/json" \
  -d '{
    "aadhaar_number": "123456789012",
    "name": "John Doe"
  }'
Retrieve Data
# Get data
curl http://localhost:8000/vault/<TOKEN>

# Get masked
curl http://localhost:8000/vault/<TOKEN>/masked
Check Health
curl http://localhost:8000/health
